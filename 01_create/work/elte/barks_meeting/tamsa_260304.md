## Polar
- emberen is mérni - szinkronnak IMU. Van a decathlonosban is?
**Why it fits your use case**

- **ECG-based HR** (electrode chest strap)
    
- **accelerometer inside the sensor**
    
- **Bluetooth Low Energy**
    
- widely used in **open-source physiology projects**
    

Hackability:

- BLE **Heart Rate Service** easily readable in Python
    
- many projects read it via **bleak / bluepy**
    
- Polar also exposes **additional vendor characteristics**
    

Typical price: **€70–90**

This is honestly still the **most hackable consumer ECG strap** available.

Wahoo TICKR X

Attach a tiny IMU board to the strap and read it separately.

Typical modules:

- **MPU‑6050**
    
- **BMI160**
    
- **LSM6DS3**
### 2️⃣ Fully custom chest node

Some labs simply build a tiny board with:

- ECG chip (e.g. **Analog Devices AD8232**)
    
- IMU
    
- ESP32 / nRF52
    

Then reuse **standard chest electrodes or strap contacts**.

This is more work but gives **raw ECG + IMU + BLE**.

### 3️⃣ A trick many researchers use

If the subject is **lying still**, respiration can often be extracted from **ECG alone**:

**ECG-Derived Respiration (EDR)** via:

- R-wave amplitude modulation
    
- baseline drift
    
- RR variability
