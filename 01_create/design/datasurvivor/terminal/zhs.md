# Directory (dir) commands



| Action                              | Command                      | Notes                                                     |
| ----------------------------------- | ---------------------------- | --------------------------------------------------------- |
| Create new directory                | mkdir dirname                |                                                           |
| Enter a directory                   | cd path/to/dir               |                                                           |
| The refer to your current directory | . (eg. cp . newlocation)     | Yes, just a dot.                                          |
| The directory above the current one | .. (eg. ../rest/of/the/path) |                                                           |
| Rename a directory                  | mv oldname newname           | Yes, it technically moves it from one place to the other. |
| delete directory (empty)            | rmdir dirname                |                                                           |
| delete directory (not empty)        | rm -rf dirname               |                                                           |
